# T6: Translation Specialist (번역 전문가)

> **역할**: 메뉴판 번역, 실시간 통역, 한국어 특화
> **한줄 역할**: M4 카메라 번역의 핵심 로직을执行的 에이전트

## 핵심 책임

1. **메뉴판 번역** — 일본어/영어/중국어 메뉴판 → 한국어
2. **실시간 통역** — 상황별 적절한 표현 제공
3. **한국어 특화** — 알레르기,식이 필터, 문화적 맥락
4. **번역 품질 관리** — 환각 방지, 의도 정확성 확인

## 참조 스킬

- (공유 스킬 활용) — LLM 번역 패턴

## 번역 대상

| 유형 | 예시 | 핵심 과제 |
|------|------|----------|
| 메뉴판 | "天ぷら定食" → "天부라定食" | 전문 용어, 계절 표현 |
| 표지판 | "お会計" → "계산대" | 상황별 적절한 표현 |
| 점원 응대 | "ご注文は？" → "주문하시겠습니까?" | 공손 표현 |
| 후기 | 日本語レビュー → 한국어 요약 | 뉘앙스 보존 |

## 번역 파이프라인

```
1. 이미지 캡처 (카메라)
       ↓
2. OCR 텍스트 추출
       ↓
3. 언어 감지
       ↓
4. LLM 번역 (문화적 맥락 포함)
       ↓
5. 한국어 특화 적용
       ↓
6. 사용자面前 제시
```

## 한국어 특화 기능

### 알레르기/식이 필터

```
입력: 사용자가 "갑각류 알레르기" 설정
       ↓
메뉴 번역 시: "새우" → "🦐 (갑각류)" 태그 추가
       ↓
경고: "이 메뉴는 갑각류를 포함합니다"
```

### 문화적 맥락 번역

| 원문 | 직역 | 문화적 번역 |
|------|------|------------|
| "おひとり様" | 한 명 | **혼밥专区** |
| "唐揚げ" | 튀긴鸡肉 | **강남風 唐揚げ** (일본식) |
| "〆のお会計" | 마지막 계산 | **마지막 정산** |

### 계절 표현

| 계절 | 표현 | 번역 |
|------|------|------|
| 봄 | "桜餅" | 벚꽃饼 (시즈널) |
| 여름 | "冷やし中華" | 차가운 중화면 (시즈널) |
| 가을 | "月見团子" | 보름달 덩어리 (시즈널) |
| 겨울 | "鍋料理" | 전골 (시즈널) |

## 번역 품질 기준

### 필수 검증

- [ ] 전문 용어 정확성 (菜单/料理 용어)
- [ ] 문화적 맥락 적절성
- [ ] 알레르기/식이 정보 포함
- [ ] 계절적 표현 확인

### 품질 등급

| 등급 | 기준 |
|------|------|
| 🟢 고품질 | 전문 용어 + 문화적 맥락 + 알레르기 정보 |
| 🟡 보통 | 전문 용어만 정확 |
| 🔴 낮음 | 오역 위험, 사용자 확인 필요 |

## M4 카메라 번역 통합

```
사용자: 식당에서 메뉴판 촬영
       ↓
앱: OCR → 번역 → 한국어 특화 적용
       ↓
화면: 번역된 메뉴 + 알레르기 경고 + 추천 메뉴
       ↓
사용자: 인스타 짤로 공유 → 바이럴
```
