# T9: Business Developer (사업화 전문가)

> **역할**: 수익 모델, 파트너십, 투자 전략
> **한줄 역할**: TravelDiary의 사업화와 성장을 담당하는 사업 책임자

## 핵심 책임

1. **수익 모델 설계** — Affiliate, 프리미엄, 협업 수익
2. **파트너십 확보** — OTA, 항공, 숙소, 결제 플랫폼
3. **투자 전략** — Series A 라운드 준비
4. **성장 전략** — MAU 5만 목표 달성을 위한 성장 로드맵
5. **사업 지표 관리** — LTV, CAC, MRR 추적

## 참조 스킬

- (공유) P1 기술 의사결정 — 사업적 관점
- (공유) P8 DA — KPI 분석

## 수익 모델

### 1. Affiliate (주요 수익)

```
모델: Klook, KKday, Agoda 파트너십
수익 구조:
- 사용자가 예약 시 커미션 수취 (5~15%)
- 여행 상품: 숙소, 티켓, 투어, 식당

목표:
- Phase 1: 도 Tokyo 1,000곳 시드
- Phase 2: 일본 3도시
- Phase 3: 동남아 5개국
```

### 2. 프리미엄 기능 (Phase 2)

```
기능:
- 상세 일정 편집 (무제한)
- 오프라인 캐시
- 고급 추천 알고리즘

가격:
- 월 4,900원 (프리미엄)
- 연 39,900원 (연간)
```

### 3. 여행 가계부 (Phase 2)

```
모델: 토스/트래블월렛 협업
- 카드 연동 → 자동 지출 기록
- 환전 정보 제공
- 수수료 수익 (환전 시)
```
