## 기능 우선순위 (Tier 시스템)

### 🟢 Tier 1 — MVP 출시 필수

```
1. AI 일정 생성 + 추천 근거 패널 (M1)
2. 환각 차단 검증 레이어 (5단계)
3. 동선 최적화 + 지도 뷰
4. Live Replan 엔진 (M3)
5. 메뉴판 카메라 번역 (M4)
```

### 🟡 Tier 2 — 출시 후 1~2개월

```
6. 한국어 후기 통합 요약
7. 예약 딥링크 (Klook/KKday/Agoda)
8. 카카오톡 일행 공유 + 동시 편집
```

### 🔴 Tier 3 — 의도적으로 미루기

```
❌ 자체 결제·예약 시스템
❌ 항공권 검색 엔진
❌ 커뮤니티/배낭톡
❌ 여행 가계부·환전
❌ 오프라인 모드
```

## 사용자 시나리오

### 시나리오 1: 첫 여행 계획

```
1. 온보딩 (4단계) → 사용자 정보 수집
2. 목적지 선택 → 도 Tokyo 선택
3. 기간/동반자 입력 → 4박 5일, 친구와
4. 일정 생성 중 → AI가 DAG 기반 일정 생성
5. 일정 상세 → 근거 패널 확인 (M1)
6. 여행 확정
```

### 시나리오 2: 여행 중 변경

```
1. D-Day 자동 전환 → 여행 중 모드 (M2)
2. 지각/변경 발생 → Live Replan 트리거 (M3)
3. 3가지 옵션 제시 → 추천/안전/강행
4. 사용자 결정 → DAG 재구성
5. 변경 저장 + Audit Log
```

### 시나리오 3: 메뉴판 번역

```
1. 식당 방문 → 메뉴판 촬영
2. OCR + 번역 → 한국어 특화 (M4)
3. 알레르기 경고 → 갑각류 알레르기 표시
4. 메뉴 선택 → 일정 추가
```
