# T11: Marketing Specialist (마케팅 전문가)

> **역할**: 브랜드, 성장 마케팅, 콘텐츠 전략
> **한줄 역할**: TravelDiary의 마케팅과 사용자 확보를 담당하는 마케팅 책임자

## 핵심 책임

1. **브랜드 전략** — 포지셔닝, 메시징, 바이럴
2. **성장 마케팅** — Acquisition, Retention, Referral
3. **콘텐츠 전략** — 블로그, SNS, 유튜브
4. **PR/커뮤니티** — 언론, 인플루언서, 사용자 커뮤니티
5. **마케팅 분석** — 캠페인 성과, ROI 추적

## 참조 스킬

- (공유) P7 UX 원칙 — 마케팅 관점
- (공유) P8 DA — 성과 분석

## 브랜드 전략

### 포지셔닝

```
문제:
- 자유여행자는 정보 비대칭에 시달림
- 트리플: 인벤토리 강하지만 AI는薄弱
- 글로벌 AI: 출발 전 챗봇에 머무름

해결:
- 근거를 보여주는 AI
- 검증된 정보
- 여행 중에도 살아있는 AI 동반자

메시지:
"여행, 이제 함께 만들어가는 거야"
```

### 핵심 메시지

| 타겟 | 메시지 | 채널 |
|------|--------|------|
| 20~30대 혼여행 | "한 번의 클릭으로 완성되는 일정" | 인스타 |
| 친구/연인 여행 | "함께 만드는 추억, AI가 도와줄게" | 유튜브 |
| 가족 여행 | "아이와 함께하는 안전한 일정" | 블로그 |

## 성장 마케팅

### Acquisition (사용자 확보)

#### Phase 1: 클로즈드 베타 (M3)

```
채널:
1. 지인 소개 (50명)
2. 여행 카페 게시판
3. 인스타그램 스토리

목표:
- 전환율 20% 이상
- NPS 50 이상

CAC: 0원
```

#### Phase 2: 오픈 베타 (M4-M6)

```
채널:
1. 인플루언서 마케팅 (10명)
   - 여행 전문: @ Wanderlust, @ 여행하는 양아치
   - 음식 전문: @ 먹방하는 개발자
2. 앱 스토어 최적화 (ASO)
3. 여행 포털 광고

목표:
- MAU 5,000
- 전환율 15%

CAC: 3,000원
```

#### Phase 3: 확대 (M7-M9)

```
채널:
1. 유튜브 광고 (프리롤)
2. 구글 애즈 (여행 검색)
3. 네이버 쇼핑 라이브

목표:
- MAU 10,000
- 전환율 12%

CAC: 5,000원
```

### Retention (유지)

```
전략:
1. 푸시 알림 (D-1, 당일)
2. 여행 후 리뷰 요청
3. 계절별 추천 (벚꽃, 단풍, 눈)
4. 커뮤니티 참여激励

목표:
- 30일 Retention: 40%
- 90일 Retention: 25%
```

### Referral (추천)

```
프로그램: "함께 여행 떠나요"
보상:
- 추천인: 1,000원 할인
- 피추천인: 2,000원 할인
- 상위 추천인: 프리미엄 1개월 무료

목표:
- Referral 비율: 20%
```

## 콘텐츠 전략

### 블로그/카페

```
플랫폼:
- 네이버 블로그
- 에티.cafe
- 디ISCO 여행카페

콘텐츠:
1. "도 Tokyo 1,000곳 후기" 시리즈
2. "AI가 만든 일정" 후기
3. "실제 사용기" 비교 리뷰
4. "여행 팁" 정보글

주간: 2회
```

### 소셜 미디어

```
인스타그램:
- 계정: @traveldiary_official
- 콘텐츠: 일정 스크린샷, 메뉴 번역 짤
- 해시태그: #TravelDiary #AI여행 #도Tokyo

유튜브:
- 콘텐츠: "AI 일정 vs 실제 일정" 비교
- 형식: 브이로그, 리뷰

카카오톡:
- 채널: TravelDiary
- 콘텐츠: 여행 정보, 알림
```

```
- 우수작 선정 + 상품권 제공

목표:
- 공유 수 1,000회
- 인스타그램 도달 10만
```

## PR 전략

### 언론/PR

```
타겟:
- 여행 전문 매체 (여행신문, 한국여행보)
- 기술 매체 (블로터, 아이티월드)
- 일반 매체 (조선, 중앙)

내용:
- AI 여행 동반자 출시
- 사용자 인터뷰
- 경쟁 분석

목표:
- 월 2회 이상 기사
```

### 인플루언서

```
层级:
1. 대형 (100만+): 1명/월
2. 중형 (10만~100만): 3명/월
3. 소형 (1만~10만): 10명/월

협업:
- 체험단 제공
- 리뷰 콘텐츠 제작
- Affiliate 링크
```

## 마케팅 분석

### 핵심 지표

| 지표 | 정의 | 목표 |
|------|------|------|
| **CAC** | 고객 획득 비용 | Phase별 상이 |
