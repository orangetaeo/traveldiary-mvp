**날짜**: 2026-04-29
**참석**: R1·R3·T7·T17·T16·T12·T13
**관련 ADR**: ADR-014

## 합의

### A1. 데이터 흐름
- `lib/mode-transition.ts` (순수): `calculateDDay(startDate, now)`, `isWithinBoundary(loc, code)`, `detectMode(trip, now, loc?)`.
- 푸꾸옥 경계 상수: 중심 `(10.225, 103.96)`, 반경 30km.
- /itinerary/[id]엔 "여행 중 모드로 전환 (데모)" 버튼 → `/travel/<id>?demo=in`로 navigation.
- /travel/[id]은 Server에서 시드 → `<TravelHome>` Client wrapper.

### A2. UI (T17)
- `<TravelHeader>`: "DAY n · HH:MM" 형식 (1분마다 setState로 시간 갱신).
- 진행률 카드: 오늘 일정 중 종료 시각이 현재 이전인 항목 / 전체.
- 현재 일정 카드 (raised, 코랄 강조): 시간·이름·근거 첫 줄 + "지금 진행 중" 펄스 점.
- 다음 일정 카드 (plain): 시간·이름·이동 시간 안내.
- FAB: 카메라 번역 (M4 사이클 4까지 disabled), 주변 검색 (사이클 4까지 disabled). 디자인만 노출.
- 색상: `[data-travel-mode="in-travel"]` → `--color-mode-primary: #FF6B47` (코랄, accent).

### A3. CSS 변수 (globals.css)
```css
:root {
  --color-mode-primary: #5B4BC4;  /* 보라 (pre-travel) */
}
[data-travel-mode="in-travel"]   { --color-mode-primary: #FF6B47; }
[data-travel-mode="post-travel"] { --color-mode-primary: #1D7F5C; }
```

### A4. 시연 시나리오 (T12 골든패스)
- /itinerary/demo-trip-phu-quoc → 푸터 위 "여행 중 모드 (데모)" 버튼 → /travel/demo-trip-phu-quoc.
- 헤더: "DAY 1 · 현재시각" — Day는 시드 startDate(2026-05-14) 기준이지만 데모 모드라 dayIndex=0(=Day 1)로 표시.
- 진행률 카드 + 현재 일정 (Day 1 첫 항목 = 호텔 체크인) + 다음 일정 (즈엉동 야시장).
- FAB(코랄 + 보조 흰색) — 카메라 번역 디자인 노출 (사이클 4 활성화).
- 색상: 헤더와 강조 텍스트가 보라 → 코랄로.
